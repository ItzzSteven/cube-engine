# Cube Engine (cubity)

## Description

cube engine is a unity inspired engine to program games in python

![Innovative Widget Screenshot](cubity.png)

## Features

*   **3d rendering (cubegl):** for 3d projects on cubity
*   **2d game support:** make 2d games
*   **CubityScript:** adds functions for 2d and 3d game development
